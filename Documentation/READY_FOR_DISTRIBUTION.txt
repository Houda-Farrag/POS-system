## 💾 Data Isolation

### Each PC has:
- Its own database (pos.db)
- Its own backups folder
- Independent users and settings
- **Data does NOT automatically sync**

### To share data between PCs:
1. Go to Admin → Backup → Create Backup
2. Save backup file to USB/cloud
3. On other PC: Admin → Restore → Select backup file
4. Done!

(Full sync coming in Phase 2)

## 🆘 If Something Goes Wrong

### "App won't start"
1. Close all instances
2. Try running as Administrator (right-click → Run as admin)
3. Wait 5 seconds
4. Try again

### "Database is locked"
1. Close the application
2. Wait 5 seconds  
3. Start again
4. If persists: Delete `src/db/pos.db` file (app recreates it)

### "Node.js not found"
1. Download from https://nodejs.org/ (LTS version)
2. Install with default settings
3. **Restart computer**
4. Try again
